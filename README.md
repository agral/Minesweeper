# Minesweeper

A minesweeper game in C++ utilizing the SDL2 library.  
_In development_
It will be gradually developed in order to create an AI solver
of Minesweeper games.

Features to be added:
 * Game logic
 * Text interface
 * Graphical interface
 * AI solver
